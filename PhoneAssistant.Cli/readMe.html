PhoneAssistant Command Line Interface (CLI) Documentation

CLI Overview

The PhoneAssistant Command Line Interface (CLI) provides a method for bulk operations associated with managing mobile phones and tablets.

Command Structure

CLI command structure consists of the cli application ("pac"), the commands ("base", "disposal"), and possibly command arguments and options.

.\pac base .\pac disposal --folder c:\temp

base Command

Imports an EE base report into the PhoneAssistant database.

Usage

.\pac base [-file|-f] <IMPORT_FILE_PATH>

Options

--file or -f

**Required.** Full path to the EE base report file to import (must be an `.xlsx` file).
